Instance Methods

numberOfOverlaysInOverlayView:

Returns the number of overlays managed for anOverlayView by the data source object.

- (NSUInteger)numberOfOverlaysInOverlayView:(AZOverlayView *)anOverlayView

Parameters

anOverlayView

The overlay view that sent the message.

Return Value

The number of overlays in anOverlayView.

Discussion

Returns the number of overlays managed for anOverlayView by the data source object.

An instance of AZOverlayView uses this method to determine how many overlays it should create and display. Your numberOfOverlaysInOverlayView: implementation can be called very frequently, so it must be efficient.
